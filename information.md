# ملخص مشروع شركة القديراني (Next.js)

هذا المشروع عبارة عن موقع تعريفي باللغة العربية لشركة القديراني للدواجن، مبني بتقنيات حديثة ويركز على عرض الأقسام الأساسية للشركة (من نحن، المنتجات والخدمات، القيم، الشراكات، والتواصل). تم تصميم الواجهة باتجاه يمين-إلى-يسار مع دعم الوضع الداكن.

## التقنيات والاعتمادات
- Next.js `16` (نمط App Router) مع React `19`.
- Tailwind CSS `v4` عبر إضافة PostCSS `@tailwindcss/postcss`.
- أيقونات MUI (`@mui/icons-material`) مع Emotion مثبتين.
- TypeScript مفعّل، مع وجود مزيج من ملفات `.tsx` و`.jsx`.
- تحميل خط محلي (Expo Arabic) باستخدام `next/font/local`.
- إعداد الصور البعيدة عبر `next.config.ts` للسماح بمصادر Google (`lh3.googleusercontent.com`).

## هيكل المشروع
- `src/app/layout.tsx`: يحدد الهيكل العام للصفحة، يضبط `lang="ar"` و`dir="rtl"`، ويحمّل الخط المحلي ويضيف شريط التنقل.
- `src/app/page.tsx`: الصفحة الرئيسية التي تُركّب الأقسام التالية:
  - `components/navbar/page.jsx`: شريط تنقل ثابت مع قائمة للجوال وزر وضع داكن (يبدّل صنف `dark` على عنصر `html`).
  - `components/hero/page.tsx`: قسم بطولي بصورة خلفية ورسائل ترحيبية وروابط دعوة للإجراء.
  - `components/about/page.jsx`: قسم تعريفي مع صورة بعيدة ونقاط مميّزة وإحصائيات وأزرار تواصل.
  - `components/products/ProductsSection.tsx`: شبكة منتجات/خدمات (٨ بطاقات) بصور بعيدة ووصف مختصر وزر عرض التفاصيل.
  - `components/WhyUs/page.jsx`: قسم القيم ولماذا تختار الشركة.
  - `components/partner/page.jsx`: قسم الشراكات مع دعوة للبدء.
  - `components/footer/page.jsx`: تذييل يحوي معلومات العنوان والهاتف والبريد وروابط شبكات التواصل.
- `src/app/globals.css`: تعريف سمات وألوان عبر متغيرات CSS وتنسيق الوضع الداكن باستخدام متغيرات Tailwind v4 (`@theme inline`).
- `public/`: صور الأيقونات والخطوط (`Fonts/ExpoArabic`).
- ملفات الإعداد:
  - `next.config.ts`: ضبط `images.remotePatterns` لتحميل الصور من Google.
  - `tsconfig.json`: مسار `@/*` إلى `src/*` وخيارات TypeScript الصارمة.
  - `eslint.config.mjs`: إعداد ESLint مع قواعد Next.js (Core Web Vitals + TypeScript).
  - `postcss.config.mjs`: تفعيل إضافة Tailwind v4.

## الوظائف الأساسية
- واجهة RTL عربية كاملة مع ضبط اللغة والاتجاه من الجذر.
- تبديل الوضع الداكن عبر زر في الـ Navbar (تبديل صنف `.dark` على عنصر `html`).
- أقسام محتوى ثابتة تعرض نبذة، منتجات/خدمات، قيم الشركة، الشراكات، ومعلومات اتصال.
- تحميل صور من مصادر خارجية (Google) بالإضافة إلى صور محلية ضمن `public/`.

## التشغيل والبناء
- `npm run dev`: تشغيل خادم التطوير على `http://localhost:3000`.
- `npm run build`: بناء الإنتاج.
- `npm run start`: تشغيل البناء المنتج.
- `npm run lint`: فحص الشفرة عبر ESLint.

## ملاحظات عامة
- لا توجد مسارات API أو منطق خلفي؛ الموقع ثابت يغلب عليه العرض التعريفي.
- الاعتمادات المثبتة تشمل `@mui/material` و`@emotion`؛ الاستخدام الحالي يتركز على أيقونات MUI فقط ويمكن توسيع الاستفادة لاحقًا.
- توجد مزاوجة بين TypeScript وJavaScript؛ يمكن توحيد النمط مستقبلاً إن لزم.
